# Expense Calculator in Python

## Project Overview

This is a simple **Expense Calculator** made with Python.
It helps you **track daily expenses**, see **how much you spend in each category**, and **manage your money better**.
All expenses are saved in a file, so you don’t lose your data.

---

## Features

Add an Expense: Quickly add things you buy, like food, transport, or bills.
View All Expenses: Check all your recorded expenses at a glance.
Check Totals by Category: Know exactly how much you spend on different categories.
Delete an Expense: Remove entries if you made a mistake.
Data Persistence: All expenses are saved automatically in a file.

---

## How It Works

When you start the program, it loads your previous expenses from a file if available.
You will see a menu with options to add, view, or delete expenses, check totals, or exit.
Adding an expense is simple: enter the name, category, and amount. The program records the date automatically.
You can view your expenses and see the total money you’ve spent.
You can check totals by category to understand your spending habits.
If you make a mistake, you can delete any expense easily.
